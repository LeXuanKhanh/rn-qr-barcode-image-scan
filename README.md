# rn-qr-barcode-image-scan

react native library for scanning qrcode/barcode from image

## Installation

```sh
npm install rn-qr-barcode-image-scan
```

## Usage


```js
import { multiply } from 'rn-qr-barcode-image-scan';

// ...

const result = await multiply(3, 7);
```
